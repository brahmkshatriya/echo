TrackerClient

interface TrackerClient : ExtensionClient

Used to track the playback of a track.

You can override the following methods to get the track details:

  • onTrackChanged

  • onPlayingStateChanged

Can be implemented by both:

  • MusicExtension

  • TrackerExtension
